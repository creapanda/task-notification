# Task Notification Application Roadmap

## Goal

Build a Java desktop application that allows users to create tasks, set reminders, receive desktop notifications, organize tasks by priority, and track completed tasks.

## Planned Features

- Create, edit, and delete tasks
- Set due dates and reminder times
- Receive desktop notifications
- Organize tasks by priority
- Mark tasks as completed
- View pending, overdue, and completed tasks

## Suggested Screens

- Dashboard
- Task list
- Add/edit task form
- Completed tasks
- Settings

## Development Phases

1. Project structure
2. Basic JavaFX window
3. Task model and database access
4. Task creation and list display
5. Completion tracking
6. Reminder scheduler
7. Desktop notifications
8. Packaging
