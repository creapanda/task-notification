# Task helper description

# GOAL
Task Helper is a desktop application designed to help users organize and manage their tasks. It provides real-time deadline notifications to ensure important tasks are completed on time while offering a simple and user-friendly interface.

# function

- Display upcoming tasks along with their deadlines.
- Create, edit, and delete tasks.
- Mark tasks as completed or incomplete.
- Export completed tasks to an Excel spreadsheet.
- Send desktop notifications when a task is approaching its deadline.
- Run automatically on system startup and continue running in the background to monitor task deadlines.
